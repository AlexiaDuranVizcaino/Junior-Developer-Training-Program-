# SAP CPQ Junior Developer Training Program

This repository contains the training modules for the **SAP CPQ Junior Developer Program**, designed to build practical skills in Python, JavaScript, REST APIs, DevOps, and SAP CPQ scripting.

## 📌 Program Overview

**Target duration**: 80–100 hours  
**Target profile**: Junior developer customizing SAP CPQ using Python (scripts in events), JavaScript/Knockout (front-end templates), integrating REST APIs, and participating in deployment (DevOps) workflows.

## 📚 Modules

- **Python OOP + Clean Code + Testing** — Core foundation for writing clean, modular scripts for SAP CPQ events.  
- **JavaScript + KnockoutJS + DOM** — Front-end customization in templates with MVVM bindings.  
- **REST APIs (Postman + FastAPI)** — Integrating with back-end APIs, testing and debugging.  
- **DevOps (Git, GitHub, CI)** — Version control, branching, CI/CD basics for safe deployments.  
- **Software Design & Patterns** — Improve design decisions, modularity, and architecture.  
- **SAP CPQ Learning Journey & Scripting** — Official content aligned with SAP CPQ internals.

Each module includes 1–2 core courses and sometimes optional extras. The selected courses are
free and include hands-on exercises or labs (not just theory).


### 📊 Overview: Modules, Key Courses, Estimated Hours

1. **Python OOP · Clean Code · Testing**  
   **Goal:** Write maintainable, testable scripts  
   **Main Courses:** Great Learning (OOP) + freeCodeCamp (Pytest) + Exercism  
   **Hours:** 12–15  

2. **JavaScript + DOM + KnockoutJS**  
   **Goal:** CPQ front-end templates using MVVM  
   **Main Courses:** freeCodeCamp JS + Knockout interactive + The Odin Project DOM  
   **Hours:** 16–24  

3. **REST APIs (Postman + FastAPI)**  
   **Goal:** Integrate and test APIs  
   **Main Courses:** Postman Academy + freeCodeCamp FastAPI  
   **Hours:** 10–14  

4. **DevOps (Git, GitHub, CI)**  
   **Goal:** Version control, automation, deployments  
   **Main Courses:** Udacity Git + GitHub Skills + Microsoft Learn Actions  
   **Hours:** 10–14  

5. **Software Design & Patterns**  
   **Goal:** Better modularity and architecture  
   **Main Courses:** Udacity Design of Computer Programs + optional Architecture course  
   **Hours:** 8–12  

6. **SAP CPQ (official) + scripting**  
   **Goal:** Master SAP CPQ templates & events  
   **Main Courses:** SAP Learning Journey + Scripting Docs  
   **Hours:** 7–8  

**Total estimated:** 63–87 hours (can extend to 80–100 hours with extra exercises)
